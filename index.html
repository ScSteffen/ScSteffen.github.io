Dynamical Low Rank Training
for deep neural networks Github

How to enable automatic model compression during training without additional computational expense? We leverage low rank dynamical systems to create factorized neural networks with low memory footprint, and fast inference and training times.

High performance simulation
of particle transport Github

From clean energy over aerospace to cancer treatment, fast and memory efficient simulation of radiation transport is crucial. We leverage structure preserving neural networks to build fast and robust surrogate models for particle simulation.

A coorperation with Cory Hauck of the Oak Ridge National Lab.

KiT-RT
HPC Radiation Therapy framework Github

The KiT-RT (Kinetic Transport Solver for Radiation Therapy) framework is a high-performance open source C++ based platform for radiation transport.

Its main focus is on radiotherapy planning in cancer treatment and investigation of various research questions in the field of radiative transfer. This goal is supported by an easily extendable code structure that allows for straightforward implementation of additional methods and techniques.

Continuum breakdown prediction:
Neural network based flow-field classification Github

Gas dynamic simulations spanning multiple flow regimes are challenging. Shock regions require expensive, high resolution kinetic schemes, but are local phenomena. Affordable continuum simulation is not feasible in these shock regions. We build a physics informed neural network based detector of shock regions to enable state-of-the-art hybrid numerical simulation of difficult flow simulation.

Aerodynamic shape optimization:
Optimization with turbulent, unsteady flows Github

Drag reduction of airplane wings is crucial for fuel efficient flight. We use windowing regularization to build a robust PDE constrained optimization for unsteady flows. The method is embedded in the open-source, high performance multi-physics software SU2.

This work was awarded 1st place at the Multidisciplinary Design Optimization Studen Paper Competititon at the AIAA aviation forum 2020.
